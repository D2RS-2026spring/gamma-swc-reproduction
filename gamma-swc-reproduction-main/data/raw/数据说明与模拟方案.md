# 论文原始数据说明与文件清单
## 1. 数据来源
本文件夹包含论文《Field Testing of Gamma-Spectroscopy Method for Soil Water Content Estimation in an Agricultural Field》的全部公开原始数据，与论文数据完全一致，来源：https://figshare.com/articles/dataset/Data_From_Gamma-Spectroscopy_Method_for_Soil_Water_Content_Estimation_in_an_Agricultural_Field/25723959

## 2. 数据文件清单（共6个，按论文分析逻辑排序）
| 文件名                | 数据用途                          | 核心字段（含单位）                                                                 |
|-----------------------|-----------------------------------|-----------------------------------------------------------------------------------|
| CalibrationDataK40.csv| 40K放射性计数与土壤含水量校准数据  | date(日期)、k40_counts(次/分钟)、swc_gravimetric(g/g)、depth_cm(土壤深度，cm)      |
| CalibrationDataVeg.csv| 植被覆盖校准数据                  | date(日期)、veg_index(植被指数)、swc_gravimetric(g/g)、crop_type(作物类型)         |
| CombinedTimeSeries.csv| 土壤含水量+40K计数+植被指数时间序列| date(日期)、k40_counts(次/分钟)、swc_gravimetric(g/g)、veg_index、crop_type        |
| ArithmeticAvg.csv     | 土壤含水量算术平均值数据          | date(日期)、swc_avg(g/g)、depth_layer(土层，如0-10cm)、crop_type                  |
| V_Wt_AvgPore.csv      | 孔隙加权平均含水量数据            | date(日期)、swc_pore_avg(g/g)、pore_size(孔隙大小，μm)、depth_cm                  |
| SOCandLatticeValues.csv| 土壤有机碳(SOC)与晶格参数数据     | depth_cm(土壤深度)、soc_gkg(有机碳含量，g/kg)、lattice_spacing(晶格间距，Å)       |

## 3. 数据预处理说明
所有数据均为论文原始格式，未做修改，后续分析脚本将：
1. 处理日期格式（统一为YYYY-MM-DD）
2. 去除无效值（如k40_counts=0的异常数据）
3. 按作物类型/土层深度分组，匹配论文分析维度
